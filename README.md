# Pharma24 - Online Pharmacy Website

Pharma24 is a modern online pharmacy web application built using HTML, CSS, and JavaScript.

## Features

- User Login System
- Responsive Design
- Medicine Product Catalog
- Product Discounts & Offers
- Secure Payment Interface
- UPI, Card & Net Banking UI
- Modern Animations
- Local Storage Authentication

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
